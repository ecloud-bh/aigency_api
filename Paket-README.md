# Aigency API Client Library

Bu kütüphane, Aigency API ile etkileşim kurmanızı sağlar.

## Kurulum

Paketi yüklemek için aşağıdaki pip komutunu kullanın:

pip install dist/aigency_api-0.1.0-py3-none-any.whl

## Kullanım

Komut satırında:

aigency_api
